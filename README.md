# postsop

Monorepo root for the `postsop` React Native app and backend, managed with `pnpm` workspaces and Turborepo.

## Common Commands

```sh
pnpm build
pnpm lint
pnpm type-check
pnpm test
pnpm contracts:watch
```

## Shared Packages

- Shared package conventions live in `packages/shared/PACKAGE_TEMPLATE.md`.
- `@postsop/shared-contracts` is the reference implementation for new shared packages.
